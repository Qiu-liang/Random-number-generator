# 随机数生成器

一个基于Flask的现代化随机数生成器Web应用，支持多种分布类型和高级功能。

## 功能特性

- 生成指定范围内的随机浮点数
- 支持多种分布类型：均匀分布、正态分布、指数分布、泊松分布
- 固定种子功能，支持可重复生成
- 可调节小数精度（1-10位）
- 实时统计信息：平均值、标准差、最小值、最大值、中位数、范围
- 历史记录自动保存到本地JSON文件
- 支持导出历史记录为CSV文件
- 亮色/暗色主题切换
- 现代化响应式Web界面（苹果液态玻璃效果）

## 安装与运行

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 启动应用：
   ```bash
   python app.py
   ```

3. 打开浏览器访问：http://localhost:5000

## 使用说明

### 基本生成
1. 在"生成随机数"区域设置：
   - 最小值：随机数范围的下限
   - 最大值：随机数范围的上限
   - 数量：要生成的随机数个数（1-100）

2. 点击"生成随机数"按钮

### 高级选项
- **分布类型**：选择不同的随机数分布算法
  - 均匀分布：所有数值在范围内均匀分布
  - 正态分布：数值呈钟形曲线分布
  - 指数分布：数值呈指数衰减分布
  - 泊松分布：数值呈泊松分布

- **小数精度**：设置生成数字的小数位数（1-10位）

- **固定种子**：输入固定种子值，可重复生成相同的随机数序列

### 统计信息
生成随机数后，页面会显示实时统计信息：
- 平均值、标准差、最小值、最大值、中位数、范围

### 历史记录
- 所有生成记录自动保存
- 支持导出为CSV文件
- 可清空所有历史记录

## 技术栈

- 后端：Python Flask

- 数据存储：JSON文件
- 科学计算：NumPy

## 注意事项

- 历史记录最多保存50条
- 默认小数精度为6位
- 应用仅限本地使用，不建议在生产环境部署
- 导出的CSV文件包含完整的生成参数和统计信息
