# 随机数生成器

一个基于Flask的现代化随机数生成器Web应用，支持生成浮点数随机数并保存历史记录。

## 功能特性

- 生成指定范围内的随机浮点数
- 支持批量生成（1-100个）
- 历史记录自动保存到本地JSON文件
- 现代化响应式Web界面
- 支持清空历史记录

## 安装与运行

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 启动应用：
   ```bash
   python app.py
   ```

3. 打开浏览器访问：http://localhost:5000

## 使用说明

1. 在"生成随机数"区域设置：
   - 最小值：随机数范围的下限
   - 最大值：随机数范围的上限
   - 数量：要生成的随机数个数（1-100）

2. 点击"生成随机数"按钮

3. 生成的随机数会显示在"生成结果"区域

4. 历史记录会自动保存在页面下方

5. 可以点击"清空历史"按钮清除所有历史记录

## 技术栈

- 后端：Python Flask
- 前端：HTML5, CSS3, JavaScript
- 数据存储：JSON文件
- 样式：现代化渐变设计，响应式布局

## 注意事项

- 历史记录最多保存50条
- 随机数精度为6位小数
- 应用仅限本地使用，不建议在生产环境部署
